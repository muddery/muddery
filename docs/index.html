欢迎访问 Muddery 官网¶

Muddery 是一个用Python编写的在线文字游戏（如MUD）框架，所有的代码都是开源的，采用BSD许可证发布。

源码地址：

  • https://github.com/muddery/muddery

  • https://gitee.com/muddery/muddery

旧版文档请见这里：

https://www.muddery.org/v0.7/
